# Together

## Details
Together is a two-player top-down real-time action adventure game made in Godot and Piskel. You play as the archer Ryl and the wizard Elvyria, fighting through enemies to get to the Sacred Gates. As you progress, you gain upgrades like the Tri Bow (shoot 3 arrows at once) and the Armor of Thorns (touching enemies hurts them).
